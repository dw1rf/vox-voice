# Vox — голосовой ввод и ИИ-автопилот

Whisper GPU + Claude/Ollama/Groq на Windows. Диктовка, голосовые команды, ИИ-агент, Clipboard AI.

Один инструмент вместо VoiceAttack. Локально, офлайн, на RTX 4070 Ti (и любой CUDA-видеокарте).  
Обучать по голосу **не нужно** — модель large-v3 понимает русский из коробки.

---

## Возможности

| | |
|---|---|
| **Диктовка** | Зажми клавишу — говори — отпусти. Текст вставляется в любое окно через буфер обмена. |
| **Голосовые команды** | «открой ютуб», «сделай громче» и любые свои. Многошаговые действия как в VoiceAttack. |
| **Fuzzy matching** | Команды срабатывают при похожих фразах (порог настраивается). |
| **Автопилот (ИИ-агент)** | «Клод, открой ютуб и сделай погромче» — LLM сам разбирает просьбу и выполняет действия. |
| **Clipboard AI** | Выдели текст → нажми F9 → полупрозрачное меню над курсором: перевод, перефраз, объяснение, краткое изложение, исправление, перевод на EN. |
| **TTS-ответы** | Автопилот может озвучивать текстовые ответы через SAPI5 (Elena/Irina). |
| **История** | Вкладка с последними 500 фразами и поиском. |
| **Импорт/экспорт команд** | Бэкап и обмен наборами команд через JSON. |
| **Webhook action** | Команда может слать HTTP POST (Home Assistant, умный дом и т.д.). |
| **Красивый UI** | Тёмный графитовый интерфейс (pywebview + Edge WebView2), иконка в трее, синглтон. |

## Быстрый старт

**Рекомендуется:** двойной клик по **`WhisperUI.bat`** — откроется тёмный дашборд.

**Консольный режим:** двойной клик по **`dictate.bat`**.

> Права администратора не нужны. Первый запуск грузит модель 5–20 с.  
> Если в приложение, запущенное «от администратора», текст не вставляется — запусти Vox тоже от админа.

## Clipboard AI

Самая удобная фича для работы с текстом:

1. Выдели любой текст в любом приложении
2. Нажми **F9** (настраивается: Настройки → «Горячая клавиша Clipboard AI»)
3. Над курсором появится полупрозрачное меню:
   - Перевести на русский
   - Перефразировать
   - Объяснить
   - Краткое изложение
   - Исправить грамматику
   - Перевести на английский
4. Нажми нужное → LLM обработает → «Вставить» заменит выделение, «Копировать» положит в буфер

Использует тот же LLM что автопилот (Ollama или Groq).

## Автопилот

Скажи **«Клод, …»** и обычную просьбу — ИИ сам поймёт что делать:

- «Клод, открой ютуб и сделай погромче»
- «Клод, найди погоду в Москве»
- «Клод, запусти блокнот и напечатай привет мир»

Обычная диктовка и словарь команд работают как раньше — автопилот срабатывает только на слово-обращение.

**Провайдеры:**
- **Ollama** — полностью офлайн на видеокарте. `ollama pull qwen2.5:7b`
- **Groq** — быстрое облако, нужен бесплатный API-ключ на groq.com

**Инструменты агента:** открыть сайт, поиск Google, запустить программу, нажать клавиши, напечатать текст, HTTP webhook.

## Голосовые команды

Скажи фразу — выполнится действие. Всё остальное — диктовка.

### Команды по умолчанию

| Фраза | Действие |
|-------|----------|
| открой клод / чат / ютуб / яндекс / гитхаб | открыть сайт |
| открой эдж / блокнот / проводник | запустить программу |
| сделай громче / тише / без звука | громкость |
| пауза / следующий трек | медиа |
| закрой / сверни / переключи окно | окна |
| скопируй / вставь / отмени | Ctrl+C / V / Z |
| сделай скриншот | Win+Shift+S |

Добавить свои — вкладка «Команды» в UI. Поддерживаются многошаговые действия (url → wait → keys → text) как в VoiceAttack.

### Fuzzy matching

Если фраза не совпала точно — Vox ищет похожую команду (rapidfuzz). Порог (0–100%) настраивается в Настройках. В журнале появится `~fuzzy: «открой поромче» → «сделай громче» (87%)`.

### Типы действий

| Тип | Описание | Пример |
|-----|----------|--------|
| `url` | Открыть сайт | `https://youtube.com` |
| `run` | Запустить программу | `notepad`, `msedge` |
| `keys` | Нажать клавиши | `ctrl+s`, `volume up` |
| `text` | Вставить текст | `alexdefult@gmail.com` |
| `wait` | Пауза (мс) | `500` |
| `webhook` | HTTP POST | `https://ha.local/api/...` |

## Интерфейс

- **Дашборд** — индикатор микрофона (серый → красный → оранжевый → зелёный), статистика, живой журнал
- **Команды** — таблица команд, импорт/экспорт JSON
- **Автопилот** — настройки LLM-агента, провайдер, модель, TTS-ответы
- **История** — последние 500 фраз с поиском
- **Настройки** — горячая клавиша диктовки, Clipboard AI клавиша, модель, язык, fuzzy порог

---

## Настройки — `config.json`

```jsonc
{
  "model": "large-v3",           // Whisper модель
  "device": "cuda",              // cuda | cpu
  "hotkey": "f7",                // клавиша диктовки
  "clipboard_hotkey": "f9",      // клавиша Clipboard AI
  "fuzzy_threshold": 80,         // порог fuzzy matching (%)
  "language": "ru",
  "beam_size": 1,
  "autopilot": {
    "enabled": true,
    "provider": "ollama",        // ollama | groq
    "model": "qwen2.5:7b",
    "wake_words": ["клод", "слушай"],
    "tts_enabled": false         // озвучивать ответы агента
  }
}
```

## Стек

- **Whisper** — `faster-whisper` (large-v3, CUDA float16)
- **Аудио** — `sounddevice`
- **Горячая клавиша** — WinAPI `GetAsyncKeyState`
- **Вставка текста** — буфер обмена + Ctrl+V
- **UI** — `pywebview` + Edge WebView2 + HTML/CSS/JS (шрифты Unbounded + Onest)
- **Трей** — `pystray` + `Pillow`
- **Fuzzy** — `rapidfuzz`
- **Clipboard AI** — `tkinter` (отдельный subprocess)
- **TTS** — Windows SAPI5 через PowerShell (Elena/Irina)
- **Python** — 3.12 в `venv\` (системный Python не затронут)
